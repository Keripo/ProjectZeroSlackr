●作品正式名称
　ひぐらしのなく頃に〜鬼隠し編〜

●概要
　物語をＰＣ上で、音楽やイラストを交えながら読んでいくサウンドノベルです。
　国語力不足の稚拙な作品ですので、寛大な心でお読みいただければ幸いです☆
　
●推奨動作環境
　OS:Windows 95/98/Me/XP
　CPU:Pentium2 400MHz 以上
　メモリ:128MB 以上
　HDD:400MB 以上の空き容量
　ディスプレイ:640×480 FullColor 表示可能な機種
　サウンド:DirectSound に対応したサウンドカード

●免責
　不慮の事故などの発生によりお使いの機器にいかなる影響を及ぼした場合でも、
　07th Expansionでは一切責任を取れません。ご了承下さい…。
　（そんな物騒なことはないとは思いますが…）

●インストール
　圧縮ファイルをフォルダごと解凍して下さい。
　「ひぐらしのなく頃に.exe」を実行すれば、ゲームをはじめる事が出来ます☆

　＜注意＞
　β版等の、旧バージョンの『ひぐらしのなく頃に』がインストールされている場合は、
　同じフォルダ内にインストールしないようにして下さい。
　セーブデータ等の絡みで不正動作が起こる可能性があります。
　正常な動作のためにも、旧バージョンのフォルダは削除されることをお薦めします☆

●アンインストール
　フォルダごとゴミ箱へ捨てて下さい。

●音源について
　音楽・効果音はwave形式で再生されています。
　音が割れているように感じる時は、オーディオのプロパティ等で調節してみて下さい。
　なので、一度インストールすればＣＤは不要です☆

●転載・頒布について
　この度のシナリオは将来発表予定の『ひぐらしのなく頃に』の内の一本です。
　つまり先行体験版・テストプレイ版となっています。
　ひとりでも多くの方のご意見を頂戴できれば、これほど嬉しいことはありません。
　ですので、データの転載、頒布はこれを禁止しません☆
　その代り……ご感想を送っていただければとてもうれしいです☆
